# it-support-scripts

    A collection of basic Linux troubleshooting scripts and notes created to practice entry-level IT support tasks.

    Overview

    This repository contains small scripts and reference notes focused on common support tasks such as checking system information, reviewing disk usage, testing network connectivity, and verifying service status.

    The goal is to build practical, hands-on skills that align with help desk and junior IT support responsibilities.

    Contents

    - System information script
    - Disk usage check script
    - Network connectivity test script
    - Service status check script
    - Troubleshooting reference notes

    Skills Demonstrated

    - Linux command line
    - Bash scripting
    - Basic networking
    - System troubleshooting
    - Technical documentation

    Purpose

    I built this repository to strengthen my troubleshooting workflow and create a small set of reusable tools for common support checks in Linux environments.

    Status

    Active project. I plan to keep adding scripts and notes as I continue learning and practicing IT support concepts.
